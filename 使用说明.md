# 批量导入工具 - 使用说明

## 📦 文件说明

本工具包含以下文件：
- **批量导入.exe** - 主程序可执行文件
- **价格.json** - 价格配置文件
- **links/** - 链接数据文件夹（包含各站点的链接文件）
- **使用说明.md** - 本说明文档

## 🚀 快速开始

### 1. 准备工作
- 确保已打开"批量导入"窗口
- 确保 Excel 配置文件（属性配置.json）已准备好
- 确保 links 文件夹中有对应站点的链接文件

### 2. 运行程序
1. 双击运行 `批量导入.exe`
2. 程序会自动读取配置并执行批量导入
3. 等待程序自动处理完成

### 3. 查看结果
- 程序会在控制台输出详细的执行日志
- 检查是否有错误提示
- 在"批量导入"窗口中查看导入结果

## 📋 功能说明

### 主要功能
- ✅ 自动选择站点和店铺
- ✅ 自动导入模板和设置尺寸图
- ✅ 自动填写字段（基础加价、运费、热搜词、链接等）
- ✅ 自动处理类目修改
- ✅ 自动点击导入按钮
- ✅ 支持批量处理多条数据

### 站点切换逻辑
- 站点切换时：自动取消全部勾选，然后选择新店铺
- 店铺切换时：只取消旧店铺，勾选新店铺

### 数据处理
- 链接数据从 `links/站点/采集词_*.txt` 文件自动读取
- 价格配置从 `价格.json` 文件读取
- 类目配置从 Excel 文件读取

## ⚙️ 配置文件

### 价格.json
包含各站点的价格计算公式，格式如下：
```json
{
    "价格": {
        "台湾": {
            "0.1": "160%+3+10",
            "0.2": "160%+7+10"
        }
    }
}
```

### links 文件夹结构
```
links/
├── 台湾/
│   ├── 采集词1_时间戳.txt
│   └── 采集词2_时间戳.txt
├── 泰国/
│   └── ...
└── ...
```

### 属性配置.json
Excel 配置文件，包含：
- 站点
- 店铺
- 采集词
- 基础加价
- 运费
- 热搜词
- 类目（一级、二级、三级、四级）

## ⚠️ 注意事项

1. **运行前必须打开"批量导入"窗口**
2. **程序运行期间不要操作鼠标键盘**，避免干扰自动化流程
3. **确保配置文件格式正确**，特别是 Excel 文件
4. **首次运行可能被杀毒软件拦截**，需要添加信任
5. **确保 links 文件夹中有对应的链接文件**

## 🖥️ 系统要求

- Windows 7 及以上版本
- 无需安装 Python 或其他依赖
- 需要管理员权限（某些操作可能需要）

## 🔧 故障排除

### 问题：提示"未找到'批量导入'窗口"
**解决**：确保窗口已打开且标题完全匹配"批量导入"

### 问题：提示"未找到弹出窗口"
**解决**：检查店铺选择下拉框是否正常显示，可能需要等待更长时间

### 问题：程序无响应
**解决**：
- 检查窗口是否被遮挡
- 确保程序可以访问窗口
- 查看控制台输出的错误信息

### 问题：被杀毒软件拦截
**解决**：
- 添加信任或临时关闭杀毒软件
- 将程序添加到白名单

### 问题：链接文件未找到
**解决**：
- 检查 links 文件夹结构是否正确
- 确保文件名格式为：`采集词_时间戳.txt`
- 确保站点名称与配置中的站点名称一致

### 问题：店铺选择失败
**解决**：
- 检查店铺名称是否完全匹配
- 确保店铺在下拉列表中可见
- 如果是站点切换，程序会自动取消全部勾选

## 📝 日志文件

程序运行时会生成 `batch_import.log` 日志文件，包含详细的执行记录，可用于排查问题。

## 💡 使用技巧

1. **批量处理**：在 Excel 中配置多条数据，程序会自动按顺序处理
2. **站点切换优化**：相同站点的数据会批量处理，减少重复操作
3. **店铺切换优化**：相同店铺的数据会批量处理，提高效率
4. **断点续传**：如果程序中断，可以重新运行，程序会从上次停止的地方继续

## 📞 技术支持

如遇到问题，请检查：
1. 控制台输出的错误信息
2. batch_import.log 日志文件
3. 配置文件格式是否正确
4. 窗口是否正常打开
