# Excel级联下拉菜单使用说明

## 功能说明

本脚本将 `类目文本.json` 文件中的数据转换为 WPS 表格（或 Excel）的级联下拉菜单。

## 使用方法

### 方法一：直接运行脚本

```bash
python create_cascading_dropdown.py
```

### 方法二：指定输入和输出文件

```bash
python create_cascading_dropdown.py "类目文本.json" "输出文件名.xlsx"
```

## 输出文件说明

生成的 Excel 文件包含两个工作表：

### 1. 数据输入（主工作表）

- **A列**：一级类目（下拉菜单）
- **B列**：二级类目（根据A列选择自动更新）
- **C列**：三级类目（根据A、B列选择自动更新）
- **D列**：四级类目（根据A、B、C列选择自动更新）

### 2. 类目数据（辅助工作表）

包含所有类目数据和查找表，用于支持级联下拉菜单功能。
- **可以隐藏此工作表**，但**不要删除**，否则级联下拉菜单将无法正常工作。

## 使用步骤

1. **打开生成的Excel文件**（使用WPS表格或Microsoft Excel）
2. **在"数据输入"工作表中**，从第2行开始输入数据
3. **选择一级类目**：点击A列单元格，从下拉列表中选择
4. **选择二级类目**：选择一级类目后，B列会自动更新可选项
5. **选择三级类目**：选择二级类目后，C列会自动更新可选项
6. **选择四级类目**：选择三级类目后，D列会自动更新可选项

## 工作原理

级联下拉菜单使用以下技术实现：

1. **命名区域**：为每个类目组创建命名区域
2. **VLOOKUP函数**：根据上级类目查找对应的命名区域名称
3. **INDIRECT函数**：动态引用命名区域
4. **数据验证**：设置下拉列表的数据源

## 注意事项

1. **文件路径**：确保 `类目文本.json` 文件与脚本在同一目录
2. **文件格式**：JSON文件必须是UTF-8编码
3. **工作表保护**：不要删除"类目数据"工作表
4. **数据完整性**：确保JSON文件格式正确，包含完整的层级结构

## 故障排除

### 问题1：找不到JSON文件
- 确保 `类目文本.json` 文件与脚本在同一目录
- 检查文件名是否正确（包括扩展名）

### 问题2：级联下拉菜单不工作
- 确保"类目数据"工作表没有被删除
- 检查命名区域是否正确创建
- 尝试重新生成Excel文件

### 问题3：某些类目没有下拉选项
- 检查JSON文件中该类目是否有子级
- 如果某个类目没有子级，对应的下拉菜单将为空

## 依赖项

- Python 3.6+
- openpyxl >= 3.0.0

安装依赖：
```bash
pip install openpyxl
```
