# 批量导入工具 - 打包说明

## 📦 打包步骤

### 1. 运行打包脚本
双击运行 `build_batch_import.bat`，脚本会自动：
- 检查并安装 PyInstaller
- 安装依赖项
- 检查必要文件
- 执行打包

### 2. 打包输出
打包完成后，在 `dist` 目录下会生成：
- **批量导入.exe** - 主程序可执行文件
- **价格.json** - 价格配置文件
- **links/** - 链接数据文件夹
- **使用说明.md** - 使用说明文档

## 📋 打包文件清单

### 必需文件
- ✅ `run_complete.py` - 主程序入口
- ✅ `价格.json` - 价格配置文件
- ✅ `links/` - 链接数据文件夹（包含所有子文件夹和文件）
- ✅ `使用说明.md` - 使用说明文档

### 自动包含的模块
- `batch_import_automation/` - 批量导入自动化模块
- `category_automation/` - 类目修改自动化模块
- 所有 Python 依赖库

## 🚀 部署到其他电脑

### 方法一：直接复制 dist 目录
1. 将 `dist` 目录下的所有文件复制到目标电脑
2. 确保目标电脑已打开"批量导入"窗口
3. 双击运行 `批量导入.exe`

### 方法二：创建安装包
1. 将 `dist` 目录下的所有文件打包成 ZIP 或 RAR
2. 在目标电脑上解压
3. 运行 `批量导入.exe`

## ⚙️ 配置文件说明

### 需要用户自己准备的配置文件
- **属性配置.json** - Excel 配置文件（需要用户根据实际情况配置）
  - 这个文件不会被打包，需要用户自己准备
  - 程序会在运行时读取这个文件

### 已打包的配置文件
- **价格.json** - 价格计算公式（已打包）
- **links/** - 链接数据文件夹（已打包）

## 📝 注意事项

1. **打包前检查**
   - 确保 `价格.json` 文件存在
   - 确保 `links` 文件夹存在且包含数据
   - 确保 `使用说明.md` 文件存在

2. **打包后检查**
   - 检查 `dist` 目录下是否包含所有文件
   - 检查文件大小是否正常（exe 文件通常 20-50MB）
   - 测试运行是否正常

3. **部署注意事项**
   - 确保目标电脑是 Windows 系统
   - 首次运行可能被杀毒软件拦截
   - 确保目标电脑已打开"批量导入"窗口
   - 需要准备 `属性配置.json` 文件

## 🔧 打包参数说明

### .spec 文件配置
- `--onefile` - 打包成单个可执行文件
- `--console` - 显示控制台窗口
- `--hidden-import` - 隐藏导入的模块
- `datas` - 包含的数据文件

### 可执行文件特性
- 单文件打包，无需安装
- 包含所有依赖，无需 Python 环境
- 控制台模式，方便查看日志

## 📊 文件大小参考

- `批量导入.exe` - 约 30-50MB（包含 Python 解释器和所有依赖）
- `价格.json` - 约 几KB
- `links/` - 根据实际数据大小
- `使用说明.md` - 约 几KB

## 🐛 常见问题

### 打包失败
- 检查是否安装了所有依赖
- 检查文件路径是否正确
- 查看错误信息

### 打包后无法运行
- 检查是否包含所有必要文件
- 检查是否有杀毒软件拦截
- 查看控制台错误信息

### 文件缺失
- 检查 .spec 文件中的 datas 配置
- 确保源文件存在
- 重新打包
